<h1>📝 To-Do List</h1>
<p>Aplicação simples de lista de tarefas desenvolvida com <strong>HTML, CSS e JavaScript</strong>, com foco em prática de lógica e manipulação do DOM.</p>

<h2>🚀 Funcionalidades</h2>
<ul>
    <li>Adicionar tarefas</li>
    <li>Remover tarefas</li>
    <li>Armazenamento no navegador (LocalStorage)</li>
    <li>Atualização automática da lista</li>
</ul>

<h2>🧠 Objetivo</h2>
<ul>
    <li>Manipulação do DOM</li>
    <li>Eventos (submit, click)</li>
    <li>Arrays e lógica de programação</li>
    <li>Uso de localStorage</li>
</ul>
